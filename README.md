# Mack Graungaard-Robinson — Portfolio

Personal portfolio site built with Next.js 14, React Three Fiber, Framer Motion, and TypeScript.

Live at: [mackgr.dev](https://mackgr.dev) *(coming soon)*

---

## Stack

- **Framework** — Next.js 14 (App Router)
- **Language** — TypeScript (strict)
- **Styling** — Tailwind CSS + shadcn/ui
- **Animation** — Framer Motion + GSAP
- **3D** — React Three Fiber + Three.js
- **Email** — Resend
- **Deployment** — Vercel

---

## Features

- R3F particle network hero with mouse-reactive repulsion
- Interactive skills constellation (SVG, category filtering)
- CLI terminal easter egg — press `~` anywhere
- MDX case studies for project deep-dives
- GitHub stats dashboard
- Contact form via Resend
